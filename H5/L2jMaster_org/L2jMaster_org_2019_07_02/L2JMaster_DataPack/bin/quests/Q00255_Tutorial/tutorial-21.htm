[Human Fighter Class Transfer]:
A Fighter who has reached level 20 may undertake a class transfer to a Warrior, a Knight or a Rogue. A character may undertake the class transfer quest at level 18; if successful, they will be able to use more specialized and powerful skills than before. In order to become a Warrior, you must first meet Master Auron. In order to become a Knight, you must first meet Sir Klaus Vasper. And in order to become a Rogue, you must first meet Captain Bezique. All can be found in Gludin Village. Think carefully about which class you want to select, then seek out the appropriate NPC.
