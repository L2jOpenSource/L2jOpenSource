[Human Mystic Class Transfer]:
A Human Mystic who has reached level 20 can change their class to a Wizard or a Cleric. A character may undertake the class transfer quest at level 18; if successful, they will be able to use more specialized and powerful skills than before. In order to become a Wizard, you must first meet Parina, and in order to become a Cleric, you must first meet Priest Zigaunt. They can both be found in Gludin Village. Think carefully about which class you want to select, then seek out the appropriate NPC.
